# Security Policy

## 지원 버전 (Supported Versions)
현재 개발 중인 메인 브랜치에 대해서만 보안 업데이트를 지원합니다.

| 버전 | 지원 여부 |
| ------- | ------------------ |
| main (최신) | :white_check_mark: |
| < 1.0 | :x: |
